# TestiFlow Studio 用户手册

## 1. 项目介绍

TestiFlow Studio 是一个基于Vue 3的智能视频生成平台，旨在提供一个直观、高效的视频内容创作环境。通过本平台，您可以轻松完成从文案编辑、提纲生成到最终视频渲染的全流程操作，大幅提升视频内容创作效率。

### 1.1 核心功能

- **提纲生成**：根据标题和主要内容自动生成结构化视频提纲
- **脚本编辑**：基于提纲智能生成脚本，支持自定义编辑
- **视频生成**：将脚本转换为高质量视频，支持多种分辨率和格式
- **历史记录**：管理和查看已生成的视频和脚本
- **个性化配置**：支持暗黑模式和API密钥管理

## 2. 环境要求与安装

### 2.1 系统要求

- **Node.js**: 16.0 或更高版本
- **npm**: 7.0 或更高版本
- **浏览器**: 推荐使用Chrome、Firefox、Edge最新版本

### 2.2 安装步骤

1. 克隆项目仓库：
```bash
git clone [项目地址]
cd testiflow-studio-frontend
```

2. 安装依赖：
```bash
npm install
```

3. 启动开发服务器：
```bash
npm run dev
```
项目将在 http://localhost:3000 启动

4. 生产环境构建：
```bash
npm run build
```

5. 预览构建结果：
```bash
npm run preview
```

## 3. 快速入门

### 3.1 初始配置

首次使用TestiFlow Studio，您需要进行以下配置：

1. **API密钥设置**：
   - 在系统设置中找到API密钥配置选项
   - 输入您的API密钥
   - 系统会自动保存并使用该密钥进行后续请求

2. **主题设置**：
   - 点击界面右上角的主题切换按钮选择明亮或暗黑模式
   - 系统会记住您的偏好设置

### 3.2 视频创作流程

完整的视频创作流程包括以下步骤：

1. **数据准备**：在数据展示页面浏览或上传素材
2. **提纲生成**：输入视频标题和主要内容，生成结构化提纲
3. **脚本编辑**：基于提纲编辑完整脚本
4. **视频生成**：配置视频参数并生成最终视频
5. **查看历史**：在历史记录页面管理已生成的视频

## 4. 功能模块详解

### 4.1 视频生成工作台

**路径**: `/`

视频生成工作台是应用的核心功能页面，集成了脚本编辑器和视频预览两个主要组件。

**使用方法**:
1. 在左侧脚本编辑区编写或粘贴视频脚本
2. 配置主题、关键词、时长和风格等参数
3. 点击生成按钮开始视频生成
4. 在右侧预览区查看生成进度和结果

**功能特点**:
- 实时显示视频生成进度
- 支持视频播放控制
- 提供错误信息展示
- 支持一键保存生成结果

### 4.2 提纲生成

**路径**: `/outline`

提纲生成页面帮助用户快速创建结构化的视频提纲。

**使用方法**:
1. 输入视频标题和主要内容
2. 点击生成按钮获取提纲
3. 编辑和调整生成的提纲
4. 点击保存或直接生成脚本

**功能特点**:
- 智能分析内容生成章节结构
- 支持手动编辑提纲内容
- 可直接从提纲生成完整脚本
- 提供提纲历史记录管理

### 4.3 脚本编辑

**路径**: `/script/:id`

脚本编辑页面提供了强大的脚本创作和编辑功能。

**使用方法**:
1. 从提纲生成脚本或直接创建新脚本
2. 编辑脚本内容和参数
3. 保存脚本或直接生成视频

**功能特点**:
- 支持从提纲自动生成脚本
- 提供脚本模板选择
- 根据主题和关键词生成文案
- 支持脚本格式化和预览

### 4.4 历史记录

**路径**: `/history`

历史记录页面展示用户已生成的视频列表，支持查看和删除操作。

**使用方法**:
1. 浏览历史生成记录
2. 点击记录查看详情
3. 重新编辑或删除历史记录

**功能特点**:
- 按创建时间排序展示历史记录
- 使用状态标签区分成功和失败的生成任务
- 支持查看详情和删除操作

### 4.5 数据展示

**路径**: `/data`

数据展示页面用于浏览和管理系统数据。

**使用方法**:
1. 浏览数据列表
2. 使用分页控件浏览更多数据
3. 查看数据详情

## 5. API接口使用

### 5.1 认证方式

所有API请求需要在HTTP头部包含认证信息：

```
Authorization: Bearer {apiKey}
```

其中`{apiKey}`为用户的API密钥，可在系统设置中配置。

### 5.2 核心API概览

#### 提纲生成

- **生成提纲**: `POST /api/generate/outline`
  - 根据标题和主要内容生成视频提纲
  - 参数：title（标题）、main_content（主要内容）

- **保存提纲**: `POST /api/outline/save`
  - 保存生成的提纲
  - 参数：title（标题）、outline（提纲内容数组）

#### 脚本生成

- **生成脚本**: `POST /api/generate/script/{outlineId}`
  - 根据提纲ID生成完整脚本
  - 参数：outlineId（提纲ID）、sections（章节内容数组，可选）

- **生成文案**: `POST /api/script/generate`
  - 根据主题和关键词生成文案
  - 参数：topic（主题）、keywords（关键词数组）、duration（时长）、style（风格）

#### 视频生成

- **生成视频**: `POST /api/video/generate`
  - 根据脚本生成视频
  - 参数：script（脚本内容）、options（视频选项，如分辨率、格式）

- **获取任务状态**: `GET /api/video/task/{taskId}`
  - 获取视频生成任务的当前状态
  - 参数：taskId（任务ID）

## 7. 常见问题解答

### 7.1 视频生成失败

如果视频生成失败，请检查：

1. API密钥是否正确设置
2. 网络连接是否正常
3. 脚本内容是否符合要求
4. 查看控制台错误信息以获取更多详情

### 7.2 页面加载缓慢

如果页面加载缓慢，可以尝试：

1. 检查网络连接
2. 清除浏览器缓存
3. 确保您的设备满足系统要求

### 7.3 API密钥配置

API密钥可以通过以下方式配置：

1. 在登录后自动获取
2. 在系统设置中手动输入
3. 通过环境变量配置（开发环境）

### 7.4 视频格式支持

系统目前支持以下视频格式：

- MP4（推荐）
- WebM
- AVI

支持的分辨率包括：720p、1080p、4K。
